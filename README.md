# KYC_Admin

## Introduction
**Admin Server** provides an Admin UI in which administrator can manage KYC Database System. Some basic functions:
* Create account (Operator, Client, User)
* Login/Logout
* Verify KYC

## Quick Start

Initialize Database (If you ensure you have ever run it, skip to the next scripts).
```
npm run db
```

Run Admin Server
```
npm install
npm start
```
